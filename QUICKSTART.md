# Quick Start - Deploy in 2 Minutes!

## Fastest Way to Deploy (No Code Required)

### Vercel - Drag & Drop

1. Go to [vercel.com](https://vercel.com)
2. Sign up with GitHub, GitLab, or Bitbucket
3. Click "Add New..." → "Project"
4. Click "Browse" and select the `score-sensei-web` folder
5. Click "Deploy"
6. Done! Your site is live 🎉

**Your URL will be**: `https://score-sensei-web-xxxx.vercel.app`

### Add Your Custom Domain (Optional)

1. Click on your project
2. Go to "Settings" → "Domains"
3. Click "Add"
4. Enter your domain (e.g., `scoresensei.com`)
5. Follow the DNS setup instructions
6. Wait 5-10 minutes for DNS to propagate
7. Your site is now at your custom domain with free SSL!

## Alternative: Netlify Drag & Drop

1. Go to [app.netlify.com/drop](https://app.netlify.com/drop)
2. Drag the `score-sensei-web` folder onto the page
3. Done! Instant deployment

**Your URL will be**: `https://random-name.netlify.app`

To add custom domain: Site Settings → Domain Management → Add custom domain

## CLI Deployment (For Developers)

```bash
# One-time setup
cd /Users/joevanderveen/Development/score-sensei/score-sensei-web
vercel login

# Deploy
vercel --prod
```

That's it! The website is now live and you can change the domain anytime from the dashboard.
